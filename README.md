# 🩺 Thyroid Cancer Recurrence Prediction using XGBoost

## 📌 Project Overview

This project builds a Machine Learning model to predict whether thyroid cancer will **recur (Yes/No)** based on patient clinical features.

The model is trained using **XGBoost Classifier** and evaluated using multiple performance metrics

The project follows a complete end-to-end ML pipeline from data preprocessing to model deployment.

## Accuracy of 97%.


## 🎯 Objective

- Predict thyroid cancer recurrence
- Build a robust ML classification model
- Evaluate performance using multiple metrics
- Save the trained model for future use

## 🛠️ Technologies & Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost
